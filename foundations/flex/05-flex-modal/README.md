# Un estilo común de "modal"

Este es otro patrón muy frecuente en la web. La solución para este ejercicio es _sencilla_... pero puede que no te resulte evidente al principio. Necesitarás editar un poco el HTML para colocar todo donde debe estar.

### Una pista
Dependiendo de cómo abordes este ejercicio, puede que necesites revisar la propiedad `flex-shrink` para evitar que un elemento flex se aplaste. Además, presta atención a la estructura del HTML: considera añadir un contenedor adicional que englobe los divs header, button, main text, cancel y continue; y considera mover el div del encabezado para que abarque también el botón.

### Comprobación

- El icono azul está alineado a la izquierda.
- Hay el mismo espacio a ambos lados del icono (las distancias entre el icono y el borde de la tarjeta, y entre el icono y el texto, son iguales).
- Hay padding alrededor del borde del modal.
- El encabezado, el texto y los botones están alineados entre sí.
- El encabezado está en negrita y con un tamaño de texto ligeramente mayor que el del texto.
- El botón de cierre está alineado verticalmente con el encabezado y situado en la esquina superior derecha de la tarjeta.
