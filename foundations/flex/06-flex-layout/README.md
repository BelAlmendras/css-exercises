# ¡Una página completa!

Flexbox es útil para maquetar páginas completas, además de los componentes más pequeños con los que ya hemos trabajado. Para este ejercicio te dejamos un poco más de trabajo, con algunas cosas que quizás no hayas visto todavía. ¡Está perfectamente aceptable buscar en Google lo que no sepas!

### Pistas
- Puede que quieras buscar algo como `CSS remove list bullets`. Ya lo hicimos en ejemplos anteriores, pero no aquí. Hora de aprender.
- Averiguar cómo dar estilo a los enlaces en CSS puede ayudarte a quitar esa molesta subrayado...
- Hemos añadido `height: 100vh` al `body`... esto hace que el body tenga exactamente la misma altura que la ventana. Para fijar el footer en el fondo necesitarás usar flex y cambiar la dirección a columna.

### Comprobación

- El header está en la parte superior de la página, el footer en la parte inferior, y permanecen en su sitio si redimensionas la pantalla.
- El header y el footer tienen padding.
- Los enlaces en el header y el footer están empujados hacia los extremos.
- Hay espacio entre los enlaces en el header y el footer.
- El footer tiene un fondo gris claro (`#eeeeee`).
- El logo, el input y los botones están centrados en la pantalla.
- Los botones tienen una cantidad adecuada de padding.
- Hay espacio entre el logo, el input y los botones.
